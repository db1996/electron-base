# Electron template
![alt text](icons/sidebar/logo-dark.svg)

Electron Base Template

## quick start

[Go to releases](https://github.com/db1996/electron-base/releases/latest) to download the latest installer.

## Features

- Autoupdater
- System tray/startup options
- Utilities for opening/saving files/folders
- Working github workflow for releases (Windows, Linux, Mac)
- ShadCN UI with a bunch of custom premade form components
- Some utility components test

## Build yourself

Make sure you have nodejs and pnpm installed

### Install dependencies

```bash
$ pnpm install
```

### Build

```bash
# For windows
$ pnpm build:win

# For macOS
$ pnpm build:mac

# For Linux
$ pnpm build:linux
```

### Development

```bash
$ pnpm dev
```
